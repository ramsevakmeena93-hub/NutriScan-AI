# NutriScan AI – Instant Food Calorie Scanner

**Scan any food in 0.5 seconds → Get calories, carbs, protein & fat instantly!**  
A smart, beautiful, and fully responsive web app built as a **Final Year B.Tech Project**.


---

### Features
- **Instant Camera Access** – One tap to open camera (works perfectly on Android & iPhone)
- **Smart AI Food Detection** – Detects Indian foods like Idli, Dosa, Samosa, Biryani, Paneer, etc.
- **Real-time Nutrition Breakdown** – Calories, Carbs, Protein, Fat with portion multiplier
- **Daily Calorie Tracker** – Visual progress bar with smart warnings
- **Late Night Alert** – Warns if eating high-carb food after 10 PM
- **Beautiful & Mobile-First Design** – Looks like a real startup app
- **No backend / No installation** – Pure HTML, CSS, JavaScript

---

### Tech Stack
- HTML5
- CSS3 (Vanilla + Flexbox + Grid)
- JavaScript (ES6+)
- Mobile-first responsive design
- Camera API (`capture="environment"`)

### How to Run Locally
1. Clone or download this repository
2. Open `index.html` in your browser  
   **For camera to work**: Use **Live Server** (VS Code) or host online
3. On mobile → Open the link → Tap **"Use Camera"** → Start scanning!

> Camera only works on **HTTPS** or **localhost** (security rule)
